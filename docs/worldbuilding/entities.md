# Entity Library

Every Worldforge world has an Entity Library — a database of all the characters, places, factions, items, species, and events that make up your world's lore. AI generates an initial set when you create your world, and you can add, edit, or remove entities at any time.

## Entity Types

There are six entity types, each with its own color-coded tag:

| Type | Description |
|------|-------------|
| **Characters** | Named individuals — heroes, villains, rulers, merchants, anyone with a story |
| **Locations** | Places in your world — cities, forests, ruins, temples, battlefields |
| **Factions** | Groups and organizations — kingdoms, guilds, cults, armies, trade unions |
| **Artifacts** | Legendary items — weapons, relics, books, crowns, magical objects |
| **Species** | Races and creatures — elves, dragons, sentient fungi, whatever your world has |
| **Events** | Historical moments — wars, discoveries, cataclysms, coronations, treaties |

## What Each Entity Has

Every entity gets a full page with:

- **Name** — the entity's title
- **Type** — one of the six types above
- **Lore** — a rich text description generated by AI or written by you
- **Custom Facts** — key-value pairs for specific details (e.g., "Age: 347", "Allegiance: The Silver Court")
- **Tags** — labels for organization and filtering
- **Connections** — relationships to other entities (see [Connections](connections.md))
- **Era** — which historical era the entity belongs to (see [Timeline](timeline.md))

## Creating Entities

### AI Generation
When you first create a world by typing a concept, AI generates a full set of entities automatically. For example, the prompt "a dying world where the last city floats above an ocean of ash" might generate:
- Characters like "Kael Ashborn" and "The Hovering Choir"
- Factions like "Songkeepers" and "Cinderwraiths"
- Locations, artifacts, species, and events to match

### Manual Creation
You can also create entities manually:
1. Go to your world's Entity Library
2. Click **New Entity**
3. Choose a type, give it a name, and write its lore
4. Add custom facts, tags, and connections

### Editing
Every field on an entity is editable. Click into any text to modify it. Changes save automatically.

## Entity Filtering

The Entity Library page has filter buttons for each type (Characters, Locations, Factions, Artifacts, Species, Events). Click a type to show only entities of that type. Entities are also filterable by era using the era filter.

## The Game Connection

Entities aren't just lore — they connect to the playable game. The game and the lore exist in the same world. NPCs you talk to in-game may reference entities from your library. The world map shows entity pins at their locations. Everything is interconnected.
