# World Map

Every world in Worldforge has an interactive world map — a bird's-eye view where you can see entity locations, draw regions, assign faction territories, and watch borders shift across eras.

## Overview

The world map is a visual canvas representing your world's geography. Entity pins mark where characters, locations, factions, and other entities exist in your world.

## Features

### Entity Pins
Every entity with a location appears as a pin on the map. Pins are color-coded by entity type:
- Characters, locations, factions, artifacts, species, and events each have their own color
- Click a pin to open that entity's details
- Hover to see a quick preview

### Regions
Draw regions on the map to define geographical areas:
- Kingdoms, territories, biomes, or any area you want to delineate
- Regions can overlap to show contested territories
- Each region can be named and described

### Faction Territories
Assign regions to factions to show who controls what:
- Color-coded by faction
- Territories can change across eras — watch borders shift as history unfolds
- Overlapping territories indicate disputed zones

### Era View
Use the era filter to see how the map changes across different historical periods. Faction borders expand, contract, and shift as events reshape the political landscape.

## Editing the Map

The map is fully editable:
- **Place pins** for new entities
- **Draw regions** with the region tool
- **Assign territories** to factions
- **Move pins** by dragging them
- **Remove pins** and regions as needed

## The Game Connection

The world map represents the same world you walk through in the playable game. The hub zone, grassland, and village all exist within this larger world. The map gives you the macro view while the game gives you the ground-level experience.
