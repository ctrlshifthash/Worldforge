# Controls

Full keyboard controls for the Worldforge game.

## Movement

| Key | Action |
|-----|--------|
| **W** / **Arrow Up** | Move up |
| **A** / **Arrow Left** | Move left |
| **S** / **Arrow Down** | Move down |
| **D** / **Arrow Right** | Move right |

Your character faces the direction you move. Movement is tile-based on a 2D canvas.

## Combat

| Key | Action |
|-----|--------|
| **SPACE** | Attack (melee, directional) |

Attacks hit in the direction your character is facing. You can dodge enemy attacks by moving out of range and kiting. Damage numbers float above enemies and the player.

## Interaction

| Key | Action |
|-----|--------|
| **E** | Interact (talk to NPCs, activate shrines, inspect points of interest) |
| **1** / **2** / **3** | Select dialogue options when talking to an NPC |

Walk up to an NPC or interactable object and press **E** to engage. Dialogue choices appear numbered — press the corresponding key to respond.

## Building (World Owner Only)

| Key | Action |
|-----|--------|
| **B** | Open/close the build menu |
| **R** | Rotate selected item (0° → 90° → 180° → 270°) |
| **Click** | Place the selected item at cursor position |

Only the world owner can build. Press **B** to open the build menu with three tabs: Structures, Props, and Decoration. Select an item, position it on walkable tiles, and click to place. Press **R** to rotate before placing.

## Camera

The camera follows your character automatically, keeping you centered on screen.
